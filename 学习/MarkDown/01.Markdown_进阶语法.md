
# Markdown 命令大全 - 进阶语法

继基础语法后，Markdown 还提供了一些进阶语法，用于创建更丰富和复杂的文档格式。

## 表格

使用管道 `|` 和连字符 `-` 创建表格。表头和单元格通过管道分隔。

```markdown
| 标题 1 | 标题 2 | 标题 3 |
| ------ | ------ | ------ |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```
### 单元格对齐

使用冒号 `:` 在表头定义对齐方式。

```markdown
| 左对齐 | 居中  | 右对齐 |
| :----- | :---: | -----: |
| 单元格 | 单元格 | 单元格 |
```

### 单元格内使用其他 Markdown

在单元格内使用列表、代码或其他 Markdown 元素。

```markdown
| 列表       | 代码         |
| --------- | ----------- |
| - 项目1   | `代码`      |
| - 项目2   | `更多代码`  |
```
## 任务列表

任务列表使用带方括号的列表项，方括号内包含空格表示未完成，`x` 表示完成。

```markdown
- [ ] 未完成的任务
- [x] 完成的任务
```

## 代码块高亮

多行代码块可以指定语言进行语法高亮。

```markdown
```javascript
console.log('Hello, world!');
```
```

## 脚注

脚注允许你在文档中添加注释和引用。

```markdown
这是一个脚注的例子[^1]。

[^1]: 这是脚注的内容。
```

## 水平分割线

使用三个或更多的连字符 `---`、星号 `***` 或下划线 `___` 创建水平分割线。

```markdown
---

***

___
```

## HTML

Markdown 支持在文档中直接使用 HTML。

```html
<div>
    <p>这是一个 HTML 段落。</p>
</div>
```

## 转义字符

使用反斜杠 `\` 转义 Markdown 特殊字符。

```markdown
\*这不是斜体文本\*
```
