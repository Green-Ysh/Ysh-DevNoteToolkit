
# Markdown 命令大全 - 基础语法

Markdown 是一种轻量级标记语言，用于创建格式化文本。以下是一些基础语法的介绍。

## 标题

使用 `#` 符号创建标题。`#` 的数量代表标题的级别。

```markdown
# 标题 1
## 标题 2
### 标题 3
#### 标题 4
##### 标题 5
###### 标题 6
```

## 段落和换行

段落之间留一个空行。行末尾添加两个空格可以创建换行。

```markdown
这是第一个段落。

这是第二个段落。  
这里是第二行。
```

## 强调（粗体和斜体）

- 使用 `*` 或 `_` 来创建斜体文本。
- 使用 `**` 或 `__` 来创建粗体文本。

```markdown
*斜体文本*
**粗体文本**
_另一种斜体文本_
__另一种粗体文本__
```

## 删除线

使用两个波浪符 `~~` 来添加删除线。

```markdown
~~这段文本将显示删除线~~
```

## 链接

使用方括号 `[]` 包裹链接文本，紧跟圆括号 `()` 包含链接地址。

```markdown
[Google](https://www.google.com)
```

## 图片

图片语法与链接相似，但前面有一个感叹号 `!`。

```markdown
![Alt 文本](图片地址)
```

## 列表

无序列表使用星号 `*`、加号 `+` 或减号 `-` 作为列表项的标记。有序列表则使用数字后跟点号。

```markdown
- 无序列表项 1
- 无序列表项 2
  - 子列表项

1. 有序列表项 1
2. 有序列表项 2
```

## 引用

使用 `>` 符号创建引用文本。

```markdown
> 这是一个引用。
```

## 代码

单行代码使用反引号 `` ` `` 包裹。多行代码使用三个反引号 ``` 包裹，并可以指定语言。

```markdown
`单行代码`

\```javascript
多行代码
\```
```
